# Proyecto de Práctica con Git y GitHub

Este repositorio contiene ejercicios prácticos para trabajar con control
de versiones usando Git y GitHub.

------------------------------------------------------------------------

## Objetivo

Practicar:

-   Creación y uso de ramas
-   Registro de cambios con commits
-   Integración de ramas en `main`
-   Sincronización entre repositorio local y remoto
-   Uso de autenticación SSH

------------------------------------------------------------------------

## Uso básico

Clonar el repositorio:

``` bash
git clone https://github.com/jmmendoza26/Desarrollo-de-Software.git
cd <repositorio>
```

Actualizar desde el remoto:

``` bash
git pull origin main
```

Enviar cambios al remoto:

``` bash
git push origin main
```
